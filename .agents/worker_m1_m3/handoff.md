## 2. Logic Chain
- **Step 1**: The user requires video relevance verification on the Combo Builder. To do this, we need to extract character keywords and matching logic. I observed that `validateVideoTitle` originally did not have a character name parameter.
- **Step 2**: I implemented `getCharacterKeywords(charName)` to parse, split, and normalize character names (like "Chun-Li" and "M. Bison") to yield space-separated, collapsed, and tokenized keyword subsets.
- **Step 3**: I updated `validateVideoTitle` to accept an optional `selectedChar` parameter. If present, it checks the video title against character keywords generated by `getCharacterKeywords`, ensuring strict validation.
- **Step 4**: To support Twitch integration, I created `extractTwitchInfo` (supporting VODs/clips hosts) and `fetchTwitchVideoTitle` (invoking the Twitch oEmbed API).
- **Step 5**: I updated `src/js/pages/builder.js` to render the format hints, validation status banners, platform icons, and confirmation checkbox. Publishing is blocked if a video URL is attached but the checkbox remains unchecked.
- **Step 6**: To guarantee reliability, I wrote unit tests (`video-validator.test.mjs`) and E2E specs (`builder-video-validation.spec.js`) to assert proper functionality of the validation flow.
